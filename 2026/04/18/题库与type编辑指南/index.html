PBTI 题库与Type描述编辑指南

项目概述

PBTI（Physicist Brain Type Indicator）是一个基于物理学家思维方式的人格测试应用。通过回答25道物理相关问题，系统会根据用户的回答计算8个维度的分数，最终确定16种人格类型中的一种。

分数设计原则

  1. 每个选项的8个维度分数总和建议在3-5分之间
  2. 同一题目中，不同选项的分数分布应有明显差异
  3. 分数值范围：0-2分（0=不相关，1=弱相关，2=强相关）
  4. 确保每个维度在整套题目中都有足够的覆盖

编辑流程

1. 题库编辑步骤

步骤1：理解8个维度

在编辑题目前，必须清楚理解每个维度的含义：

  • I (直觉派)：依赖物理图像、直觉、整体把握
  • D (推导派)：依赖数学推导、逻辑分析、逐步推理
  • T (理论家)：偏好理论研究、思想实验、理论构建
  • E (实验家)：偏好动手实验、观测验证、实践操作
  • M (微观视角)：关注原子、粒子、量子等微观尺度
  • C (宏观视角)：关注宇宙、星系、天体等宏观尺度
  • N (创新者)：倾向创新思维、非传统方法、突破常规
  • R (严谨者)：倾向严谨求证、精确计算、系统方法

步骤2：设计题目

  1. 题目主题：选择与物理学思维方式相关的主题
  2. 选项设计：每个题目提供3个选项，每个选项代表不同的思维方式
  3. 分数分配：为每个选项的8个维度分配分数（0-2分）
  4. 平衡性：确保整套题目中各个维度都有足够的题目覆盖

步骤3：分数验证

编辑完成后，需要验证：

  1. 每个维度的总分是否大致平衡
  2. 是否存在明显的分数偏向
  3. 题目间的分数分布是否合理

2. 人格类型编辑步骤

步骤1：理解4字母代码

每个代码由4个字母组成，分别对应4个维度的选择：

  • 第1位：I或D（思维方式）
  • 第2位：T或E（研究取向）
  • 第3位：M或C（关注尺度）
  • 第4位：N或R（解题策略）

步骤2：填写人格类型信息

  1. 名称：富有诗意且体现人格特点的名称
  2. 表情符号：选择有代表性的emoji
  3. 描述：1-2句吸引人的简短描述
  4. 核心特质：4个关键词，体现该人格的核心特点
  5. 优势与弱点：各4-5条，客观描述该人格的优缺点
  6. 相似物理学家：2-3位历史上或当代的物理学家
  7. 适合领域：3-5个该人格可能擅长的物理学领域
  8. 发展建议：1-2句建设性的建议
  9. 详细分析：多段落深入分析该人格的思维方式、适合的研究方向等

步骤3：保持一致性

确保同一维度的人格类型之间有明显的区别，同时保持整体风格一致。

最佳实践

题库设计最佳实践

  1. 题目多样性：涵盖物理学的不同领域（经典力学、量子力学、相对论、宇宙学等）
  2. 选项平衡：每个题目的3个选项应代表不同的思维方式组合
  3. 语言风格：使用通俗易懂的语言，避免过于专业的术语
  4. 场景设计：设计真实的物理学习或研究场景
  5. 文化适应性：考虑中文用户的思维习惯和文化背景

人格类型描述最佳实践

  1. 积极导向：即使是弱点，也要以建设性的方式描述
  2. 专业准确：确保物理学领域的描述准确
  3. 富有启发性：让用户感到有收获和启发
  4. 个性化：每个类型都有独特的描述风格
  5. 实用性：提供实际的发展建议和职业指导

测试与验证

1. 分数计算验证

编辑完成后，应进行测试验证：

  1. 随机选择答案，检查分数计算是否正确
  2. 测试边界情况（如所有题目都选同一选项）
  3. 验证16种人格类型都能被正确匹配

2. 用户体验测试

  1. 题目是否容易理解
  2. 选项是否有明显的倾向性
  3. 结果描述是否准确且有价值

常见问题与解决方案

Q1：如何设计一个平衡的题目？

A：首先确定题目要测试的主要维度，然后设计3个选项，每个选项侧重不同的维度组合。确保每个选项的分数分布合理，不会导致某个维度过度偏向。

Q2：如何为新的题目分配分数？

A：遵循以下步骤：

  1. 确定该题目主要测试哪几个维度
  2. 为每个选项的主要维度分配2分
  3. 为次要维度分配1分
  4. 不相关的维度分配0分
  5. 检查总分是否在3-5分之间

Q3：如何确保16种人格类型都有用户能匹配到？

A：通过题库设计确保每个维度都有足够的题目覆盖，并且分数分布合理。可以统计所有可能的答案组合，确保每种4字母代码都能通过某些答案组合达到。

Q4：人格类型描述应该多详细？

A：详细分析部分建议在500-800字之间，涵盖思维方式、适合领域、发展建议等方面。其他部分应简洁明了。
